## Requirement
* XAMPP 7.4.9 or later
* PHP 7.2 or later
```
drive/
├── Index[1].php
├── bootstrap.css
├── data.json
├── miney-logo.png
```

## Installation
Pindahkan semua file ke dalam folder
```
C:\xampp\htdocs\Index\index[1].php
```
Start Apache pada XAMPP Control Panel

Akses pada browser
```
localhost/drive/Index/index[1].php
```
